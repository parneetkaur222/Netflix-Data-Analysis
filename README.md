# Netflix Data Analysis

A complete end-to-end data analysis project exploring Netflix's movies and TV shows catalog — covering data cleaning, exploratory data analysis, visualization, and business insight generation.

## Project Overview

This project analyzes the Netflix Movies and TV Shows dataset to uncover patterns in content type distribution, geographic concentration, genre popularity, content growth over time, and audience targeting through ratings. The goal is to demonstrate a complete data analyst workflow — from raw data to actionable business insights — suitable for a data analyst internship portfolio.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
- Git & GitHub
- Power BI
