# Netflix Data Analysis

A complete end-to-end data analysis project exploring Netflix's movies and TV shows catalog — covering data cleaning, exploratory data analysis, visualization, and business insight generation.

## Project Overview

This project analyzes the Netflix Movies and TV Shows dataset to uncover patterns in content type distribution, geographic concentration, genre popularity, content growth over time, and audience targeting through ratings. The goal is to demonstrate a complete data analyst workflow — from raw data to actionable business insights — suitable for a data analyst internship portfolio.

## Key Insights

- Content Mix: Movies make up ~70% of the catalog compared to ~30% for TV shows, indicating a historically film-heavy content strategy.
- Geographic Concentration: The United States accounts for ~32% of all titles, with India as the second-largest contributor at ~11%.
- Growth Trajectory: Content additions grew nearly fivefold between 2016 and 2019, before slowing in 2020-2021 amid rising competition from Disney+ and Amazon Prime.
- Audience Targeting: Mature-rated content (TV-MA, TV-14) dominates the catalog, indicating a strong focus on adult and young-adult audiences.

Full analysis and recommendations are available in report/Netflix_Analysis_Report.md.

## How to Run This Project

1. Clone the repository:
git clone https://github.com/parneetkaur222/Netflix-Data-Analysis.git

2. Navigate into the project folder:
cd Netflix-Data-Analysis

3. Install required libraries:
pip install -r requirements.txt

4. Launch Jupyter Notebook:
jupyter notebook

5. Open notebooks/01_data_loading_and_cleaning.ipynb and run all cells.
